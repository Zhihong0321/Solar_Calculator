Install Eternalgy Agent OS

The chat screens now use the shared app install flow for the whole agent workspace, not a separate SolarChat installer.

How it works now

  • Use the shared install button shown in the agent app.
  • Android will show the browser install prompt when available.
  • iPhone will show an Add to Home Screen guide.
  • The install prompt hides automatically once you launch from the Home Screen.

Recommended entry point

Open the main dashboard and use the install experience there. That keeps the whole workspace under one app: Eternalgy Agent OS.
